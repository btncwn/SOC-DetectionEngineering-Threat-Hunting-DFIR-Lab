# SOC Investigations

## Overview

This section contains end-to-end SOC investigations performed using telemetry collected throughout the lab environment.

Investigations combine threat hunting, threat intelligence, DFIR techniques, and detection engineering methodologies to identify, analyze, and document suspicious activity.

## Objectives

- Perform SOC investigations
- Analyze attacker behavior
- Reconstruct attack timelines
- Document findings
- Develop investigative methodology
- Improve detection coverage
