# PowerShell Encoded Command Detection

## Sigma Conversion

The Sigma rule was converted into Splunk SPL using PySigma.

Command:

```bash
sigma convert -t splunk --without-pipeline \
11-detection-engineering-sigma/05-threat-intelligence-driven-detection/sigma-rules/powershell-encoded-command.yml
```

## Why No Pipeline Was Used

The objective was to observe the raw conversion output generated by PySigma before any field mapping transformations were applied.

Using:

```bash
--without-pipeline
```

allows detection engineers to:

* Understand Sigma translation logic
* Validate generated SPL
* Troubleshoot field mappings
* Learn Sigma internals

This approach is useful in laboratory environments and detection engineering training exercises.
